# 如何将本地文件复制到OpenOS
1. 找到存档文件夹。
2. 打开存档文件。例如，我要将本地文件复制到lab这个存档中，则需要打开"lab"文件夹。
!['教程_图1'](..\images\教程\教程_图1.png)
3. 找到opencomputers文件夹并打开。
!['教程_图2'](..\images\教程\教程_图2.png)
4. 如果你在这个存档里组装过一台计算机，你应该会在 "opencomputers"文件夹内看到一些名字由很长一串数字和字母组成的文件夹。其中，一个文件夹代表着该存档中的一个硬盘或磁盘列阵，文件夹名称则代表了对应的硬盘或磁盘列阵的地址。  
找到你希望将文件复制到的硬盘或磁盘列阵的文件夹，打开它。例如，我想要将文件复制到硬盘（地址：9ac733f1-3f7a-478e-b3d9-4408803a238f），那么我应该打开"9ac733f1-3f7a-478e-b3d9-4408803a238f"这个文件夹。
!['教程_图3'](..\images\教程\教程_图3.png)
5. 打开后，你将看到该硬盘或磁盘列阵的所有文件和目录，将你想要复制过去的文件或目录放到相应位置即可。
!['教程_图4'](..\images\教程\教程_图4.png)
